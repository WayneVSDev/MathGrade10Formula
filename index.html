Grade 10 Mathematics Formulas

1. Algebra

Quadratic Equations and Functions

Quadratic Formula: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

Vertex Form: $y = a(x - h)^2 + k$

Discriminant: $\Delta = b^2 - 4ac$

Exponential and Logarithmic Functions

Exponential Function: $f(x) = a \cdot b^x$

Logarithmic Function: $y = \log_a(x)$

Change of Base Formula: $\log_a(x) = \frac{\log_b(x)}{\log_b(a)}$

Systems of Equations

Cramer's Rule (for 2x2 systems):

$x = \frac{D_1}{D}, y = \frac{D_2}{D}$

where $D = ad - bc, D_1 = ed - bf, D_2 = af - ec$

Rational Expressions

Addition/Subtraction: $\frac{a}{b} \pm \frac{c}{d} = \frac{ad \pm bc}{bd}$

2. Geometry

Trigonometry

Sine: $\sin(\theta) = \frac{\text{opposite}}{\text{hypotenuse}}$

Cosine: $\cos(\theta) = \frac{\text{adjacent}}{\text{hypotenuse}}$

Tangent: $\tan(\theta) = \frac{\text{opposite}}{\text{adjacent}}$

Pythagorean Identity: $\sin^2(\theta) + \cos^2(\theta) = 1$

Coordinate Geometry

Distance Formula: $d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$

Midpoint Formula: $(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2})$

Slope Formula: $m = \frac{y_2 - y_1}{x_2 - x_1}$

Circles

Area: $A = \pi r^2$

Circumference: $C = 2\pi r$

Arc Length: $L = \frac{\theta}{360Â°} \cdot 2\pi r$

Three-dimensional Shapes

Sphere Volume: $V = \frac{4}{3}\pi r^3$

Cylinder Volume: $V = \pi r^2h$

Cone Volume: $V = \frac{1}{3}\pi r^2h$

Vectors

Magnitude: $|\vec{v}| = \sqrt{x^2 + y^2}$

Dot Product: $\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos(\theta)$

3. Statistics and Probability

Measures of Central Tendency

Mean: $\bar{x} = \frac{\sum x}{n}$

Median (odd n): $(n+1)/2$ th term

Mode: Most frequent value

Measures of Spread

Range: Largest value - Smallest value

Variance: $\sigma^2 = \frac{\sum(x - \mu)^2}{n}$

Standard Deviation: $\sigma = \sqrt{\frac{\sum(x - \mu)^2}{n}}$

Probability

$P(A \text{ or } B) = P(A) + P(B) - P(A \text{ and } B)$

$P(A \text{ and } B) = P(A) \cdot P(B|A)$

Conditional Probability: $P(A|B) = \frac{P(A \text{ and } B)}{P(B)}$

Combinatorics

Permutations: $P(n,r) = \frac{n!}{(n-r)!}$

Combinations: $C(n,r) = \frac{n!}{r!(n-r)!}$

4. Functions

Composite Function: $(f \circ g)(x) = f(g(x))$

Inverse Function: $f^{-1}(f(x)) = x$

5. Number and Quantity

Complex Numbers: $a + bi$

Absolute Value: $|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$

6. Calculus (Introduction)

Limit: $\lim_{x \to a} f(x) = L$

Derivative: $f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

7. Financial Mathematics

Simple Interest: $I = P \cdot r \cdot t$

Compound Interest: $A = P(1 + r)^n$

Future Value of an Annuity: $FV = PMT \cdot \frac{(1 + r)^n - 1}{r}$

Where:

  • $P$ = Principal
  • $r$ = Interest rate (as a decimal)
  • $t$ = Time
  • $n$ = Number of compounding periods
  • $PMT$ = Regular payment amount
